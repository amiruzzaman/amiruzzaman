Description of cars.csv file visualization:


This graph shows the comparison of three different origin of cars, such as, American, European, and Japanese. In addition, the graph shows how these three types of origin of cars are different from each other, in terms of average acceleration capacity, average number of cylinders, average horsepower, and average mile per gallon capacity.

From the graph, it is clear that American made cars are better in terms of average acceleration capacity. However, Japanese cars are better in terms of average mile per gallon capacity. This interactive graph include measure names on the top right corner, which can be turned-off and on see the analyzed results.

Description of cereal.csv file visualization:


In this case, I used the visualization to show cereal brand name, manufacturer, type, average calories, and average fat associated with a particular cereal brand. This visualization is helpful to get a quick glimpse of each brand.

Each block are representing a specific bran of cereal, and the size of the block is determined by the average calories, and average fat. For example, if average calorie and average fat value is higher, then the block is wider and lengthier. The color intensity for average fat is categories as 0 to 5. Where lighter color means close to 0 and darker color means close to 5.

Description of films.csv file visualization:


In this case, I used visualization to difference in type of movies: western, action, war, drama, comedy, horror, music, mystery, science fiction, etc. Each circle represents type of movie used by each groups (i.e., western, action, war, drama, comedy, horror, music, mystery, science fiction, etc.).

Average number of records are represented by the color, for example, lighter color represent small number and darker color represent higher number.

Description of grocery_stores.csv file visualization:


In this case, I used visualization to show difference in gender: male and female. Each circle represents type of payment method used by each groups (i.e., male and female). The size of the circle represents number of counts/frequency for each type and family size.

Note that highest payment types used by both groups are: cash, store card, check, debit card, and other. In addition, female group have done more shopping than male group. Different color scheme helps to separate each payment type and different size helps to understand the comparison of each payment type.

Description of mutual_funds.csv file visualization:


In this case, I used visualization to difference in types of mutual funds. For example, international stocks, large growth, large value, small growth, small value, etc. Each circle represents a company. Also, ratings of company shown in the visualization. The visualization grouped each category based on their ratings. Therefore, higher rating companies are residing in the center, and lower rating companies are residing away from the center. Companies can be separated by their color as well, where, each color represents type of company.
